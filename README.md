# Docker Demo Development

```bash
$ git --version
$ docker -v
$ docker-compose -v
$ git clone git@github.com-zyrave:zyrave/docker-demo-development.git docker-demo && cd docker-demo
$ git submodule init && git submodule update
$ git submodule foreach yarn
$ docker-compose up -d
```
